# ShareIt

**ShareIt** — сервис для аренды вещей между пользователями. Приложение позволяет добавлять собственные вещи для аренды и бронировать вещи других пользователей.

## Технологический стек
Spring Boot 3.3, Maven, PostgreSQL, Spring Data JPA, Lombok, REST, Docker

---

## Архитектура

Приложение реализовано в виде двух микросервисов:

- **Gateway** — выполняет первичную валидацию входящих запросов и маршрутизирует корректные запросы в основной сервис;
- **Server** — обрабатывает бизнес-логику, принимает REST-запросы от Gateway, взаимодействует с базой данных и возвращает результаты.

---

## Инфраструктура

Приложение разворачивается в трёх контейнерах:
- `db` — PostgreSQL;
- `server` — основной сервис;
- `gateway` — сервис валидации и маршрутизации.

Связь между сервисами и базой данных настраивается через **Docker Compose**.  
Передача запросов от Gateway к основному сервису осуществляется по REST с использованием `RestTemplate`.

---

## Работа с данными

Взаимодействие с базой данных реализовано через **Spring Data JPA**.

---

## Тестирование

Реализованы следующие виды тестов:
- unit-тесты бизнес-логики;
- интеграционные тесты с базой данных;
- тесты REST-эндпоинтов с использованием **MockMvc**;
- тесты сериализации и десериализации DTO (JSON).
